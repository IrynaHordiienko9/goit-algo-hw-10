#### У ході виконання завдання було обчислено визначений інтеграл функції $f(x) = x^2$ на інтервалі [0,2] різними методами.

__Отримані результати:__


|       Метод            |   Значення інтеграла    |
|------------------------|-------------------------|
| Аналітичний            | 2.666667                |
| SciPy quad             | 2.666666666666667       |
| Монте-Карло (NumPy)    | 2.661028                |
| Монте-Карло (вручну)   | 2.669494                |


**Аналіз:**

*   Різниця між аналітичним, чисельним (quad) та Монте-Карло результатами становить менше 0.001.
*   Це свідчить про високу точність методу Монте-Карло при великій кількості випадкових точок.
*   SciPy quad дає практично ідеальний результат з мінімальною похибкою ($\sim 10^{-14}$).
*   Метод Монте-Карло (як через NumPy, так і вручну) дає результат, який збігається з аналітичним і чисельним, що підтверджує правильність реалізації.


**Висновок:**  

Метод Монте-Карло є ефективним для обчислення визначених інтегралів. При достатній кількості точок його похибка мінімальна, а результат практично збігається з точним аналітичним та чисельним значенням.
